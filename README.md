# PCSV

:information_desk_person: 东大小微体育系跑操爬虫移动版 (*PaoCao Spider - Vue Version.*)，基于 Vue 搭建。

:wave: 解决体育系网站只能通过内网访问限制（校内 VPN 同样被屏蔽）。

## 使用说明

:point_right: 注意：**使用时须将设备连接至 seu-wlan**

为了缓解体育系服务器压力，网页内刷新数据按钮每半小时可点击一次。
